IB Terminal — Sound Files
=========================

Place your .mp3 sound files here:

  fill.mp3          - Plays when an order fully executes
  partial_fill.mp3  - Plays on each partial fill  
  disconnect.mp3    - Alert when an account loses connection
  connect.mp3       - Confirmation when account connects
  pnl_alert.mp3     - Warning when unrealized loss exceeds threshold

You can use any .mp3 files you like. Short sounds (1-3 seconds) work best.

Free sound effects can be found at:
  - https://freesound.org
  - https://mixkit.co/free-sound-effects/

The START_FRONTEND script will copy this folder into the React app automatically.
